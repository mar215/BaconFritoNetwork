Bacon Frito Network: La red social para empresas más sabrosa

Esta aplicación forma parte de la evaluación del curso de programador en java que está siendo impartido por Javier Gómez en la consultoría LUZTI, y ha sido organizado por GFT Iberia. Las funcionalidades de la aplicación se centran en:

$ Modelo de red social entre usuarios del servicio
$ Servicio de mensajería interna de tipo email
$ Simulación de pagos mediante un WebService
$ Utilización de elementos  y tecnologías relacionadas para implementar funcionalidades adicionales

Plataformas principales

El proyecto se va a desarrollar principalmente en JavaEE, utilizando servlets, páginas JSP, HTML y CSS, base de datos Oracle, JDBC para los accesos a la base de datos, WebServices (por determinar). Además se integrarán conocimientos de programación multihilo, expresiones regulares y patrones de diseño (MVC, Singleton y demás que puedan surgir por los devenires del desarrollo).

Herramientas y entornos de desarrollo

La herramienta principal de desarrollo es el Eclipse IDE Java EE. También se usará ScrumDo para documentar el uso de la metodología Scrum, y GitHub para el control del código y repositorios.
